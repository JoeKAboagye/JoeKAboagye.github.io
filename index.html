Landing Page

The goal of this project is to create a game in 3D using raycasting !
It was designed for kids so that it could improve their spatial recognition and analytical skills

Keyboard Keys for Players

  • Arrow Keys -- Player Directions
  • M -- Map
  • R -- Refresh
  • C -- Zoom Player view
  • [ -- Zoom Out in Game
  • ] -- Zoom In in Game
  • F -- Change Views (Normal view to Panoramic view and vice versa)
  • T -- Change Texture of walls and square columns
